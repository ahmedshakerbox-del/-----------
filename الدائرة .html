دائرة متناسقة الألوان

دائرة مقسمة إلى 6 أقسام بألوان غامقة وصافية، تتحرك عند المرور عليها.

<!-- HTML -->
<div class="circle"></div>

<!-- CSS -->
.circle {
  width: 320px;
  height: 320px;
  border-radius: 50%;
  background: conic-gradient(
    #ff3f34 0deg 60deg,
    #f39c12 60deg 120deg,
    #f1c40f 120deg 180deg,
    #27ae60 180deg 240deg,
    #2980b9 240deg 300deg,
    #8e44ad 300deg 360deg
  );
}

كيفية عمل الدائرة

  • استخدمنا خاصية conic-gradient في CSS لإنشاء تدرج دائري.
  • قسمنا الدائرة إلى 6 أقسام متساوية (كل قسم 60 درجة).
  • كل قسم يبدأ من زاوية وينتهي عند الزاوية التالية (0-60، 60-120، إلخ).
  • الخاصية border-radius: 50% تحول العنصر إلى دائرة.
  • الدائرة تتحرك وتكبر عند المرور عليها، مع ألوان غامقة وصافية.

كيفية استخدام هذا الكود

يمكنك نسخ كود HTML و CSS أعلاه واستخدامه مباشرة في مشروعك، مع إمكانية تعديل الألوان حسب رغبتك.
