أنواع الأزرار في النماذج HTML - اعداد /احمد شاكر

تعتبر الأزرار عناصر أساسية في أي نموذج ويب. في HTML، لدينا عدة أنواع من الأزرار لكل منها وظيفة مختلفة. تعرف على الفروقات بينها وكيفية استخدامها بشكل صحيح.

تم تصميم هذه الصفحة بواسطة أحمد شاكر وتعمل باستخدام HTML وCSS فقط دون الحاجة لـ JavaScript.

زر عادي

الزر العادي <button> أو <input type="button"> لا يقوم بأي سلوك افتراضي.

يُستخدم عادة لتنفيذ أوامر مخصصة عبر JavaScript ولا يؤثر على النموذج.

هذا الزر لا يؤثر على النموذج. في التطبيقات الحقيقية، يتم ربطه بوظائف JavaScript.
<button type="button">
  <i class="fas fa-hand-pointer"></i> زر عادي
</button>

زر الإرسال

زر الإرسال <input type="submit"> أو <button type="submit"> يرسل بيانات النموذج إلى الخادم.

يقوم بإرسال البيانات إلى الصفحة المُحددة في السمة action داخل الوسم <form>.

تم إرسال النموذج! (في التطبيقات الحقيقية، يتم إرسال البيانات إلى الخادم)
<form action="/submit" method="post">
  <input type="email" name="email" required>
  <button type="submit">
    <i class="fas fa-paper-plane"></i> إرسال البيانات
  </button>
</form>

زر الإعادة

زر الإعادة <input type="reset"> أو <button type="reset"> يعيد جميع حقول النموذج إلى قيمها الافتراضية.

لا يرسل النموذج، فقط يعيد الحقول إلى حالتها الأصلية عند تحميل الصفحة.

تم إعادة تعيين النموذج إلى القيم الافتراضية!
<form>
  <input type="text" value="نص افتراضي">
  <button type="reset">
    <i class="fas fa-undo"></i> إعادة التعيين
  </button>
</form>

ملخص أنواع الأزرار

النوع الوسم الوظيفة مثال
زر عادي <button> أو <input type="button"> لا يقوم بأي إجراء افتراضي، يستخدم للأوامر المخصصة <button type="button">نقر</button>
زر الإرسال <button type="submit"> أو <input type="submit"> يرسل بيانات النموذج إلى الخادم <input type="submit" value="إرسال">
زر الإعادة <button type="reset"> أو <input type="reset"> يعيد جميع حقول النموذج إلى قيمها الافتراضية <button type="reset">إعادة</button>
